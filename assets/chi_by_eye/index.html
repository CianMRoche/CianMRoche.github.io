χ by eye

χ by eye is an interactive tool for fitting models to data by eye. Adjust a curve to match a set of data points and build intuition for chi-squared, reduced chi-squared, goodness of fit, and parameter uncertainty. It is a browser-based way to develop a feel for statistical fitting before reaching for a solver.

This interactive tool runs in your browser. If it has not loaded, make sure JavaScript is enabled in your browser settings.
